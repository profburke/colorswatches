Module colors.css

Provides a collection of CSS color names and their RGB values.

For more details, see the documentation for colors .

Usage:

     css_colors = require 'colors.css'


 -- print out the names and RGB values
 for name, color in pairs(css_colors) do
       print(name, color:rgb())
 end


 -- or get the components individually
 goldenrod = css_colors.goldenrod
 print(goldenrod.name, goldenrod.r, goldenrod.g, goldenrod.b)


 -- or make use of color's __tostring method
 print(goldenrod)
